# fitness-ui-clone
Focus Module 14: Task UI

## Educational UI Prototype

This repository contains a **UI prototype created as part of a study exercise** for *Focus Module 14*.  
The purpose of this project is to **practice recreating and understanding user interface structures** of real-world applications.

The prototype was built using:

- HTML
- CSS
- JavaScript
- AI-assisted development tools
- Dataset:
  https://www.kaggle.com/datasets/exercisedb/fitness-exercises-dataset

---

## Design Reference

The interface design used in this prototype is **inspired by the mobile application of ACTIV FITNESS**.

Original website:  
https://www.activfitness.ch/

Original App:  
https://www.activfitness.ch/activ-fitness-app/

The UI was recreated solely as a **learning exercise**.
