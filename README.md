# Focus Module 14 — UI Task

## Educational UI Prototype

This repository contains a **UI prototype created as part of a study exercise** for *Focus Module 14*.  
The purpose of this project is to **practice recreating and understanding user interface structures** of real-world applications.

The prototype was built using:

- HTML
- CSS
- JavaScript
- AI-assisted development tools
- Dataset from: https://www.kaggle.com/datasets/exercisedb/fitness-exercises-dataset
---

## Design Reference

The interface design used in this prototype is **inspired by the mobile application of ACTIV FITNESS**.

Original website:  
https://www.activfitness.ch/

The UI was recreated solely as a **learning exercise to analyse how fitness apps organise information, workouts, and navigation patterns**.

---

## Important Disclaimer

- This project is **not affiliated with, endorsed by, or connected to ACTIV FITNESS**.
- The interface was recreated **only for educational and research purposes**.
- This prototype **does not reproduce the real application functionality**.
- The project **does not connect to any real ACTIV FITNESS systems, APIs, or services**.
- The application is **a front-end prototype only**, created purely for learning how UI systems are structured and implemented.

## Project Purpose

The main learning objectives of this exercise were:

- building **interactive UI components**
- using **AI-assisted coding workflows**
- experimenting with **fitness dataset integration**
