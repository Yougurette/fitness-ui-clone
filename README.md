# fitness-ui-clone
Focus Module 14: Task UI

## GitHub Pages deployment

This repository includes a workflow at `.github/workflows/deploy-pages.yml` that publishes the current `main` branch as a static website.

### One-time GitHub settings

1. Open **Settings → Pages**.
2. Set **Source** to **GitHub Actions**.
3. Push to `main` (or run the workflow manually in **Actions**).

After deployment, version pages are available under:

- `https://yougurette.github.io/fitness-ui-clone/index_v6_5.html`
- (or the latest default entry at `https://yougurette.github.io/fitness-ui-clone/` if configured separately)
